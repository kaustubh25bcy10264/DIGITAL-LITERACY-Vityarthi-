📌 Project Overview
This project is a structured Digital Literacy Portfolio designed to help students acquire essential digital competencies. It consists of five tasks that focus on safe technology use, professional online presence, collaboration, communication standards, and cybercrime awareness. The objective is to demonstrate responsible use of digital platforms while assisting others in achieving digital safety and efficiency.

📂 Module Summaries
Module 1: Created a digital literacy infographic to spread awareness about safe and effective use of technology.

Module 2: Built a professional digital portfolio using platforms like GitHub and LinkedIn.

Module 3: Practiced coding on Stack Overflow and created a Google Form for collaboration and data collection.

Module 4: Learned professional email writing and responsible social media communication.

Module 5: Studied cybercrime types and created awareness through a case study and safety checklist.

📑 Deliverables
Infographic (task‑1‑infographic/)

Google Form + response sheet screenshots (task‑3‑platforms/)
